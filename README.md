# zenhub-api-scripts
Scripts for automating Zenhub tasks

Setup Github personal access token with write access to private repos: https://github.com/settings/tokens
Setup a Zenhub access token: https://app.zenhub.com/dashboard/tokens

Add the tokens to the respective script variables.
Change permissions to run 
`chmod 755 ready_4_review.sh`

Then run ./ready_4_review.sh

Enter the issue number at the prompt.
